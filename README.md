# Kishore's Portfolio

A professional portfolio website for Kishore, a Lead Generation Specialist, SEO Executive, and Web Developer.

## Features
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop.
- **Dark Theme**: Modern and professional aesthetic.
- **Interactive Elements**: Smooth scrolling, typed text effects, and hover animations.
- **Contact Form**: Pre-configured to send emails via the default mail client.
- **Direct Call**: "Hire Me" button initiates a phone call.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- FontAwesome
- ScrollReveal.js
- Typed.js

## How to Run Locally
1. Clone the repository or download the files.
2. Open `index.html` in your browser.
   OR
   Run a local server (e.g., `npx http-server`).

## Deployment
This site is ready to be deployed on platforms like Netlify, Vercel, or GitHub Pages.
